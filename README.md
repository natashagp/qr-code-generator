# QRCode Generator

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/qr-code-generator.git
```

### Step 2

Change the directory to qr-code-generator

```shell
cd qr-code-generator
```

### Step 3

Install requirements

```shell
pip install -r requirements.txt
```

### Step 4

Run project

```shell
python main.py
```
